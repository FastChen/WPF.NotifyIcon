## 特点

- 在 Windows 上弹出气泡通知(Win10以上系统则是 Toast 的普通通知)。
- 使用 `Shell_NotifyIcon` 原生实现，无需第三方库支持。
- 支持 .NET 4.62 Framework - .NET 9
- 与 WinForm NotifyIcon 控件的使用类似(目前在实现完全一致使用方案)。
